# Дашборд продуктивности

Интерактивный дашборд для отслеживания продуктивности с использованием принципов ООП.

## Функциональность

### 6 виджетов:
1. **ToDo Widget** - управление задачами (добавление, удаление, отметка выполненных)
2. **Quote Widget** - мотивационные цитаты с кнопкой обновления
3. **Time Widget** - текущее время и дата в реальном времени
4. **Weather Widget** - информация о погоде (имитация API)
5. **Stats Widget** - статистика продуктивности (имитация API)
6. **Calendar Widget** - календарь на текущий месяц с навигацией

### Особенности:
- Динамическое добавление и удаление виджетов
- Независимое состояние каждого виджета
- Адаптивный дизайн (3 колонки на десктопе, 2 на планшете, 1 на мобильном)
- Анимации и плавные переходы
- Иконки Font Awesome для лучшей визуализации

## Технологии

- Чистый JavaScript (ES6+)
- ООП с использованием классов и наследования
- CSS Grid и Flexbox для адаптивной верстки
- CSS переменные и градиенты
- Модульная архитектура (ES6 Modules)
